# M-thodes-d-IA-explicable
Ce dépôt contient des codes sources pour plusieurs méthodes d’IA explicable (XAI) parmi les plus populaires. L’objectif est de fournir un ensemble d’outils permettant de mieux comprendre le fonctionnement des modèles complexes d’apprentissage automatique et d’intelligence artificielle, en offrant des explications à la fois locales et globales.
